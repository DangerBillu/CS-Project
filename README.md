# Event Management System

This is a web-based **Event Management System** designed as a 12th-grade Computer Science investigatory project.

## 🛠️ Features
1. **Create Events**
2. **Register for Events**
3. **View Upcoming Events**
4. **Manage Participants**

## 📋 Technologies Used
### Frontend
- **HTML**
- **Tailwind CSS**

### Backend
- **Python**
- **Flask**
